Cloudinary

Cloudinary is used to store uploaded images and generate urls, the call to Cloudinary is made from the react app the results are then sent to cloud vision api.

Google Vision

Cloud Vision API is used to analyze uploaded images looking for the data corresponding to the userâ€™s choice, the call to the cloud vision api is made from the server side.

React

The user interface for this application was created using react components. The functions responsible for sorting the images based on the responses from Google Cloud Vision are in the react app. This app is then built and served to the user's device.

Express

The server was written in express. The server sends the react app to the user, and handles Google Vision API calls from the user.

Node Package Manager

Node Package Manager was used to install dependencys for both the react app, and the express server.

Amazon Web Services

Our server is hosted in an Amazon Web Services Elastic Computing instance, which provides us with a fully scallable virtual machine.

Ubuntu

The virtual machine that hosts our application is an Ubuntu 18.04 operating system with node 10.15.1 installed.
